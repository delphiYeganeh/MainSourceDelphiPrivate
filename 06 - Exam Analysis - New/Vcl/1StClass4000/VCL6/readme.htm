1stClass 4000.01 Release

 

Installing the 1stClass help files – You will need to run the OpenHelp utility by clicking on Help | Customize from the Delphi or Builder IDE.  From there, click on Edit | Add Files to add the appropriate 1stClass help file: fc4000.hlp (which can be found in the corresponding Delphi/Builder help directory). You should add the file to the list of help files in the index and link tab pages. Do not add it to the contents tab page. After adding the .hlp file, click on File | Save Project to save your changes.

 

Warning – Delphi’s default help indexing is near the limits of Microsoft's WinHelp.  As a result, you may need to remove any indexes you do not regularly use do stay within the capabilities of WinHelp.  If you experience an empty list on the Index tab page of WinHelp, this indicates you need to remove some .hlp files using OpenHelp.
