تعريف طبقه بندي پيوست

تعريف طبقه بندي پيوست و اختصاص رنگ :

بر روي تصاوير پيوستي سند مي توان رنگ خاص تعريف كرد تا در هنگام جستجو تصاوير بتوان از رنگ مورد نظر استفاده كرده و تصوير

 مورد نظر را پيدا كرد.  

 

براي اين منظور ابتدا رنگ هاي مورد استفاده تعريف مي گردد,از منو مديريت سيستم گزينه تعريف طبقه بندي پيوست  را انتخاب

كرده و با زدن كليد  رنگ مورد نظر را انتخاب و شرح آن را درج  نماييد.

براي استفاده از آن هنگام پيوست تصاوير سند در قسمت گروه بندي با رنگ و قسمت انتصاب رنگ با كليك راست روي رنگهاي قابل

 استفاده مي توان گزينه انتصاب رنگ را انتخاب كرد.

 

· براي حذف رنگ انتخاب شده مي توان از گزينه حذف رنگ استفاده كرد.

· براي استفاده از رنگ هاي انتخاب شده از قسمت مشاهده استفاده مي گردد به اين صورت كه بر روي رنگ هاي موجود كليك كرده

 و  تصوير با رنگ مربوطه آن نمايش داده مي شود.
