SQL Server  2008 نصب   

 

 

   بر روي Microsoft Sql Server 2008 R2 Enterprise کليک کنيد.

 

  

 

  از منوي سمت چپ Installation  را انتخاب مي کنيم.

  از صفحه جديد بايد گزينه ي اول که نصب جديد است را انتخاب کنيم .

 

 

                       

 در اين مرحله از نصب خطاهاي ممکن که مانع نصب صحيح  SQLمي شوند توسط Setup Support Rules چک مي شوند تا در صورت

    يافتن خطا در  سيستم قبل از نصب برطرف کنيد و در صورت عدم وجود خطا در سيستم شما در شکل زير گزينه Ok را کليک کنيد.

 

 

 

       پس از مشاهده شکل زير دکمه Install را کليک کنيد.

  

    

 

   بر روي Next  کليک کنيد.

 

                                  

 

  در اين صفحه product key را وارد مي کنيم در نسخه هاي کرک شده اين فيلد پر شده است.

.

 

 

  تيک I accept ... را ميزنيم و سپس NEXT

 

 

   

     بعد از پايان نصب پيش نيازها دکمهOK فعال مي شود و کليک مي کنيم.

 

                                         

 در اين پنجره  Next را بزنيد و سپس Select All

 

 

 

    در اين پنجره در صورتي که قبلا بر روي ويندوز هيچ يک از نسخه هاي قبلي SQL تحت default instance نصب نشده اند

    گزينه   default instanceرا انتخاب کنيد و روي گزينه  NEXT  کليک کنيد.

   در صورتي که قبلا از  default instance  قبلا استفاده کرده ايدگزينه Name instance را انتخاب کنيد و در پنجره رو به روي آن نام دلخواهي

    را تايپ کنيد .

 

 

   جهت مشخص کردن Account Name  سرويسهاي انتخاب شده بر روي گزينه  Use the same account for all sql service  کليک کنيد و

   سپس در پنجره باز شده Nt AUTHORITY \NETWORK SERVICE  را وارد کنيد.

   

 

   

  مهم :مطابق شکل  collation را انتخاب کنيد .

 

 

     

     گزينه Windows authentication mode را فعال کنيد.     

     گزينه Add Curent user را کليک کرده و سپس Next  را بزنيد.

 

 

 

  گزينه Add Curent user را کليک کرده و سپس Next  را بزنيد.

   

 

 

                             

     

 در پنجره Reporting Services Configuration گزينه اول را انتخاب کنيد و بر روي Next  کليک کنيد.

  

                    

  در پنجره Ready to install بر روي Install کليک کنيد تا پنجره نهايي نصب ظاهر گردد.

 

 

 

   شکل زير پنجره نهايي نصب است که پس از ابن مرحله پيغامي جهت پايان و اعلام موفقيت نصب نشان داده مي شود که با ديدن اين

   پيغام SQL Server 2008 بر روي سيستم شما نصب شده است.

   

 

    پس از نصب برنامه sql server management studio  را از منوي start اجرا مي کنيم.

 

 

                            

 

   پس از نمايش فرم مربوطه نام سرور يا local  را انتخاب کرده کليک راست کرده و آيتم properties  را انتخاب مي کنيم.

 

    

   پس از نمايش فرم مربوطه از تب security  گزينه sql server  and windows  را انتخاب کرده و ok  را انتخاب مي کنيم.
