SQL Server 2000 On Win Server

    نصب SQL Server  2000 (ويندوز server)

 

 

   زماني که CD نرم‌افزار SQLServer را در درايو قرار دهيد فرم زير نمايش داده مي شود حال گزينه "SQLServer2000 Enterprise"  

    را انتخاب کنيد.

  

 

 

 

  

 

    در صورت مشاهده پيغام زير دکمهok  را کليک کنيد .

 

    مرحله اول نصب SQL Server فقط جهت خوش آمد گويي و دادن آگاهي از نرم‌افزاري است كه در حال نصب آن هستيد با زدن next   به  

    

    مرحله بعد برويد .

 

 

      به دليل اينكه اين نرم‌افزار قابليت نصب از روي شبكه را داراست، در اين پنجره از شما خواسته شده تا تعيين نماييد مي‌خواهيد برنامه

      را بر  روي همين دستگاه نصب نماييد و يا دستگاهي بر روي شبكه. چون ما نرم‌افزار را بر روي همين دستگاه نصب مي‌كنيم،

      گزينه Local Computer را انتخاب نموده و به مرحله بعد مي‌رويم.

 

 

 

   

    در پنجره Installation Selection گزينه اول را انتخاب،سپس برروي Next کليک کنيد.

 

                                                       

                                                                       

 

 

    پنجره زير را به دلخواه پر کنيد.

                                                                         

 

 

 

    بر روي Yes کليک کنيد.

 

                                                               

 

    اکنون Service and Client Tools  را انتخاب کنيد.

 

                                                           

 

 

    در اين پنجره Default  را تيک بزنيد و بر روي Next کليک کنيد.

 

                                                          

 

 

    حال در اين فرم گزينه Custom را فعال کرده و مکان نصب برنامه را در صورت لزوم تغيير دهيد.

 

                                                            

 

 

 

    پنجره Select Component را چک کنيد تا همه اجزاي ذکر شده تيک داشته باشد.

 

                                                          

 

    همانند شکل زير دو گزينه Use the same account for each service Auto…. و Use the Local System account را انتخاب کنيد .

 

                                                        

 

 

 

 

    درون پنجرهAuthentication Mode   ، Windows Authentication Mode را تيک بزنيد.

 

                                                          

 

 

 

   در مرحله بعد Sql Collation(Used for compatibility….  را تيک بزنيد و روي   Dictionary order , case-insensitive for use with the 1256

  کليک کنيد تا انتخاب گردد و بعد دکمه Next را انتخاب کنيد.

                                                       

 

 

  در دو پنجره بعد بر روي Next کليک کنيد.

 

                                                              

 

                                                              

 

 

   در اين مرحله،براي گزينه Per Seat for عدد 50(تعداد کامپيوتري که به SQL ارتباط دارند)را وارد نماييد .

                                        

                                                 

 

  هم اکنون برنامه در حال نصب شدن است.

                                           

  اين پنجره پايان عمليات نصب Sql server را اعلام مي کند .

                                          

   بعد از انجام عمليات فوق يکبار کامپيوتر خود را Restart کنيد و بعد از آن اعمال زير را انجام دهيد.

   پس از نصب برنامه enterprise manager  را از منوي start اجرا مي کنيم.

 

                               

    پس از نمايش فرم مربوطه نام سرور يا localرا انتخاب کرده کليک راست کرده و آيتم properties  را انتخاب مي کنيم.

 

    پس از نمايش فرم مربوطه از تب security  گزينه sql server  and windows  را انتخاب کرده و ok  را انتخاب مي کنيم.  sql server  و نرم افزار

   را ببنديد  .
