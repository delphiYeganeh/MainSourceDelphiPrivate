SQL Server 2000 On Win Xp & 7

   نصب SQL Server 2000  ( ويندوز xp و seven )

  

 

   زماني که CD نرم‌افزار SQLServer را در درايو قرار دهيد فرم زير نمايش داده مي شود حال گزينه "SQLServer2000 Enterprise"   

    را انتخاب کنيد.

  

 

 

 

  

 

    در صورت مشاهده پيغام زير دکمهok  را کليک کنيد .

 

 

   مرحله اول نصب SQL Server فقط جهت خوش آمد گويي و دادن آگاهي از نرم‌افزاري است كه در حال نصب آن هستيد با زدن next  به مرحله 

   

   بعد برويد .

 

                                                     

    

      به دليل اينكه اين نرم‌افزار قابليت نصب از روي شبكه را داراست، در اين پنجره از شما خواسته شده تا تعيين نماييد مي‌خواهيد برنامه

      را بر  روي همين دستگاه نصب نماييد و يا دستگاهي بر روي شبكه. چون ما نرم‌افزار را بر روي همين دستگاه نصب مي‌كنيم،

      گزينه Local Computer را انتخاب نموده و به مرحله بعد مي‌رويم.

 

    در اين پنجره از شما سوال مي‌پرسد كه آيا يك نسخه جديد SQL Server را نصب مي‌كنيد و يا مي‌خواهيد نسخه موجود را ارتقاء دهيد. چون  

    مايك نسخه جديد نصب مي‌كنيم همان گزينه Create a new instance را انتخاب نموده و به مرحله بعد مي‌رويم.

 

                                                          

  

  در اين مرحله، نام و نام شركت شما درخواست شده است. ورود نام شركت الزامي نيست. به مرحله بعدي مي‌رويم.

 

                                                               

                                                         

    

        در اين مرحله يک سري قوانين مربوط به نصب نرم افزار است که آن را تاييد مي کنيم .

     

                                                             

   گزينه Client tools only  را انتخاب کنيد.

                                                                

                                                          

   اگر اولين باري است که sqlserver نصب مي شود ، گزينه default  فعال است و بايد آن را انتخاب کنيد.

    اجزايي كه مايليد نصب شوند را انتخاب نماييد. تمامي تنظيمات پيش‌فرض را قبول نموده و به مرحله بعد برويد.

 

                                                                    

 

   به شما اطلاع داده مي‌شود برنامه Setup ، تمامي اطلاعات مورد نياز براي نصب SQL Server را جمع آوري نموده و آماده نصب برنامه است.

   با كليك دكمه    Next    وارد نصب برنامه شويد .

 

. هم اکنون برنامه در حال نصب شدن است 

 

 

   اين پنجره پايان عمليات نصب sql server را اعلام مي کند. 

      

 

       

  حال فايل setup.exe را از پوشه msde داخل sql server  CD  را اجراکنيد .

      

                                                             

  توجه کنيد که هنگام اجراي اين برنامه پيغام خاصي نمايش داده نشود .

 

      

حتما بعد از محو شدن پنجره فوق کامپيوتر خود را Restart   نماييد.

 

  پس از نصب برنامه enterprise manager  را از منوي start اجرا مي کنيم.

                                         

 

    سپس نام سرور يا localرا انتخاب کرده کليک راست کرده و آيتم properties  را انتخاب مي کنيم.

را انتخاب کرده و .را انتخاب كرده و نرم افزار را ببنديد ok   sql server  and windows گزينهsecurity پس از نمايش فرم مربوطه از تب
