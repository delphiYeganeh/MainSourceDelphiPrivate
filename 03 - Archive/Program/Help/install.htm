راهنما نصب نرم افزار

پس از نصب SQL Server    ،  سي دي بايگاني يگانه را در داخل دستگاه قرار دهيد و اطلاعات آن را در داخل هارد سيستم و در درايوي غير از درايو ويندوز کپي نمائيد.       

از پوشه DataFile  ، همانند شکل زير ، روي MakeSql.exe  دابل کليک کنيد.

 

 به اين ترتيب صفحه اي بصورت زير براي شما نمايش داده مي شود.

 

اگر فقط يک Instance روي SQL  نصب داريد ، در اين قسمت  کافيست تعيين کنيد که نرم افزار بايد به instance پيش فرض وصل شود که مي توان بجاي آن ، از نقطه استفاده کرد. در غير اينصورت بايد نام instance  كه مايل هستيد نرم افزار بر روي آن نصب گردد را وارد نمائيد.

 

پس از اينکه نام سرور را وارد نموديد  دکمه ساخت بانك اطلاعاتي را بزنيد. در صورتيکه عمليات با موفقيت انجام شده باشد پيغام زير نمايش داده مي شود .

 

در اين مرحله اتصال بانک شما به SQL Server   انجام شده است و بانك اطلاعاتي بايگاني در  آن ايجاد شده است .

در اين مرحله كار نصب نرم افزار كامل گرديده و نرم افزار آماده استفاده و بهره برداري مي باشد.
